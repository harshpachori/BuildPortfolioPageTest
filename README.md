Web development course practice exercise
Pete Thinkful | Artist Portfolio
A simple, responsive portfolio website for Pete Thinkful, a Denver-based artist specializing in abstract art, street graffiti, and collaborative projects.
Project Overview
This is a single-page portfolio website featuring three main sections:

About: Introduction to Pete Thinkful with background information
Portfolio: Showcase of three art projects (Abstract Red, Spiral Zany, Melted Rainbow)
Contact: Links to social media profiles

Features

Clean, centered layout with maximum content width of 600px
Responsive images with proper sizing
Smooth anchor navigation between sections
Custom styling with Google Fonts (Playfair Display and Source Sans Pro)
Hover effects on navigation links
Circular profile image with border
Semantic HTML5 structure

Technologies Used

HTML5
CSS3
Google Fonts
Normalize.css (for cross-browser consistency)
